## Ce que tu vas apprendre

Jusqu'ici, ton code s'exécutait ligne par ligne, du début à la fin. Mais dans la vraie vie, on a souvent besoin de **prendre des décisions** :

- Si l'utilisateur est connecté → afficher son profil
- Si le panier est vide → afficher un message
- Si la note est supérieure à 4 → afficher "Réussi"

C'est exactement ce que permettent les **conditions** en JavaScript.

## Au programme

Dans ce module, tu vas découvrir :

1. **`if` et `else`** — la base : exécuter du code seulement si une condition est vraie
2. **`else if`** — gérer plusieurs cas différents
3. **Les opérateurs de comparaison** — `===`, `!==`, `>`, `<`, `>=`, `<=`
4. **Les opérateurs logiques** — `&&` (ET), `||` (OU), `!` (NON)
5. **`switch / case`** — pour les cas multiples bien rangés
6. **L'opérateur ternaire** — une condition en une seule ligne

## Pourquoi c'est important

Sans conditions, un programme ferait toujours la même chose. Les conditions rendent ton code **intelligent** : il réagit différemment selon la situation.

Chaque site web, chaque app utilise des conditions — de la validation d'un formulaire à l'affichage d'un menu selon le rôle de l'utilisateur.

## Prêt ?

Les exercices qui suivent te guideront pas à pas. On commence doucement avec `if / else`, et on monte en puissance jusqu'au ternaire.
