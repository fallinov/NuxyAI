description: Révise les derniers changements du code
---

Analyse les modifications récentes du projet Nuxy.

Effectue les actions suivantes :
1. Affiche les derniers commits avec `git log -5 --oneline`
2. Affiche les fichiers modifiés avec `git status`
3. Analyse les changements avec `git diff` sur les fichiers modifiés

Puis fournis un résumé :
- Des changements effectués
- De la qualité du code
- Des éventuels problèmes détectés (sécurité, performance, bugs)
- Des suggestions d'amélioration
