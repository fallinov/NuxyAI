Exécute les tests automatisés pour valider le bon fonctionnement de la console pédagogique et du système de messages.

```bash
node test-console-pedagogique.mjs
```

Analyse les résultats et indique :
- Le nombre de tests réussis/échoués
- Les éventuelles erreurs détectées
- Des suggestions de correction si des tests échouent
