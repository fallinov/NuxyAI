---
description: Lance le serveur de développement Nuxt
---

Démarre le serveur de développement local avec hot-reload.

Exécute la commande en arrière-plan :
```bash
npm run dev
```

Attends que le serveur démarre et indique :
- L'URL d'accès (généralement http://localhost:3000)
- Le port utilisé
- Que le hot-reload est actif

Note : Le serveur restera actif en arrière-plan. Tu peux le tuer plus tard si nécessaire.
